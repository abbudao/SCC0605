# Introdução 
Repositório para a construção de um parser da linguagem Jason da disciplina SCC605

# Como reproduzir
## Requisitos
Java 8+
## Instale o Gradle
### Arch-linux 
`sudo pacman -S gradle`

# Building 
Na pasta raiz do projeto:
`gradle build`

# Testing 
Na pasta raiz do projeto:
`gradle test`

*Um report em html estará disponível após esse comando*
